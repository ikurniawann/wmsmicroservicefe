# WMS Frontend

Warehouse Management System - Frontend (Next.js)

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn/ui
- Zustand (State Management)

## Environment Variables

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=https://wms-backend.vercel.app
```

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploy ke Vercel dengan framework preset: **Next.js**
